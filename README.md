# Typing Speed Test App

This is a typing speed test application implemented in Python using the Tkinter library. Users can test their typing speed by typing a provided text, and the application will calculate the words per minute (WPM) based on their typing speed.

## Description

This application allows users to test their typing speed by typing a given sample text. The application calculates the words per minute (WPM) based on the time taken to type the text and the number of words typed.

## Installation

1. Clone this repository to your local machine using the following command:
```sh
gh repo clone aFro95/Typing-Speed-Test
```

2. Navigate to the project directory:
```sh
cd typing-speed-test
```
3. Run the Python script:
```sh
python Typing Speed Test.py
```


## Usage

1. Run the Python script in your preferred environment.
2. Type the provided sample text in the entry field.
3. The timer starts automatically once you start typing.
4. Once you finish typing the entire sample text correctly, the application will display your words per minute (WPM) score.
5. You can start a new typing test by clicking the "Start Typing" button again.
